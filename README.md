# AI Speech Intelligence System

This project implements an **AI Speech Intelligence System** that converts speech into text and generates summaries using pretrained foundation models.

## Project Overview

The system uses speech input and produces useful output in the form of **text summarization**.
It combines speech recognition and natural language processing models to demonstrate a simple multimodal AI pipeline.

All implementation code is provided in the **Google Colab notebook** included in this repository.

* FLAN-T5 (Google) for summarization
